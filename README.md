# ğŸ“‚ Installation & Setup

1. Clone the repository
   git clone https://github.com/YOUR_USERNAME/ShopGenie.git
   cd ShopGenie
2. Install dependencies
   npm install
3. Setup Environment Variables (.env)
   MONGO_URI=your_mongodb_atlas_connection_string
